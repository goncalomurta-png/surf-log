## Regras de veredicto (por ordem de prioridade)

### 1. Wave Power — factor principal
| Wave Power   | Veredicto base                                      |
|--------------|-----------------------------------------------------|
| < 2 kW/m     | ❌ Não vai — flat independentemente do resto        |
| 2–4 kW/m     | ⚠️ Marginal — shorebreak, sem pico estruturado      |
| 4–7 kW/m     | ✅ Fraco mas surfável, ideal para Tomás iniciado    |
| 7–18 kW/m    | ✅ Bom — zona ideal para ambos                      |
| 18–35 kW/m   | ⚡ Exigente — Rodrigo sim, Tomás avaliar            |
| > 35 kW/m    | ⚡ Muito exigente — Rodrigo pode, Tomás não         |

### 2. Direcção swell — ajuste ao veredicto
- **W/NW** → manter veredicto (mais favorável para costa sul)
- **W+S** → neutro
- **N/NW** → baixar um nível (swell de norte entra mal nas Milícias)
- **S** → ⚠️ vento sul + swell sul = condições pobres

### 3. Vento — ajuste ao veredicto
- **N/NE (offshore)** → melhorar um nível (limpa a onda)
- **Fraco (<10 km/h)** → neutro
- **Sul >20 km/h** → baixar dois níveis (destrói a onda)

### 4. Amplitude de maré — ajuste menor
- Marés vivas (>1.0m): avaliar estado de maré na hora (MB preferível para Milícias)
- Marés mortas (<0.5m): com WP ≥ 7 kW/m é suficiente; com WP < 5 kW/m baixar um nível
- Impacto geral: pequeno — não altera veredicto principal
